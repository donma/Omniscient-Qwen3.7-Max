## 技術棧

- **HTML5** — 語意化標籤 + Schema.org 結構化數據
- **CSS3** — CSS 變數、Grid、Flexbox、動畫、pseudo-elements
- **Bootstrap 5.3.3** — CDN 引入（Grid + 基礎排版）
- **Bootstrap Icons 1.11.3** — CDN 引入（圖標庫）
- **Google Fonts** — Rajdhani + Noto Sans TC
- **Vanilla JavaScript** — 無框架依賴

## 瀏覽器開啟方式

直接雙擊任一 `.html` 檔案即可在瀏覽器中開啟（`file://` 協定）。

> 注意：Bootstrap、Bootstrap Icons 和 Google Fonts 透過 CDN 載入，首次開啟需要網路連線。

## 設計特色

- 深藍黑科技網格配色，營造專業信任感
- Dashboard 風格 Hero 區域（CSS 繪製儀表板圖形）
- 6 大解決方案卡片含狀態條
- 數字滾動動畫（Intersection Observer）
- 完整 RWD 三斷點（992px / 768px / 576px）
- 繁體中文內容，強 SEO（Schema.org + OG tags）
