# T019 沐光家居 MUJI Living

居家佈置品牌完整網站模板。雜誌式編輯風格，暖白自然色調，以對稱 grid、數字編號與大量留白呈現靜謐優雅的居家美學。

## 設計特色

- 雜誌式編輯風格：對稱 grid、數字編號（01/02/03）、極細線框
- 暖白自然色調：#faf9f6 底色、#8b7355 自然棕、#5c4a32 深棕
- 無陰影卡片：細線框 + hover 圖片微放大
- Cormorant Garamond + Noto Serif TC 字型搭配
- 完整繁體中文選單與內容

## 頁面結構

| 頁面 | 檔案 | 說明 |
|------|------|------|
| 首頁 | index.html | 雜誌式 Hero + 精選系列 + 靈感文章 + 展示間 + 品牌故事 |
| 品牌故事 | about.html | 創辦理念、核心價值、里程碑 |
| 精選系列 | services.html | 四大系列（傢俱/燈飾/家飾/花器） |
| 產品詳情 | service-detail.html | 原木傢俱系列深入介紹 |
| 展示間 | portfolio.html | 空間佈置案例 |
| 客戶回饋 | reviews.html | 真實使用心得 |
| 常見問題 | faq.html | 配送/訂製/保養 FAQ |
| 預約參觀 | booking.html | 展示間預約表單 |
| 訂購流程 | process.html | 6 步驟流程說明 |
| 靈感文章 | blog.html | 居家佈置趨勢分享 |
| 文章詳情 | blog-detail.html | 北歐風客廳佈置專文 |
| 聯絡我們 | contact.html | 三處展示間資訊 + 表單 |

## CDN 依賴

- Bootstrap 5.3.3
- Bootstrap Icons 1.11.3
- Google Fonts: Cormorant Garamond + Noto Serif TC

## 技術規格

- CSS 變數系統（色票集中管理）
- 三斷點 RWD：1024px / 768px / 576px
- IntersectionObserver 捲動動畫
- FAQ accordion、表單假送出、平滑滾動
- Organization Schema (JSON-LD)
