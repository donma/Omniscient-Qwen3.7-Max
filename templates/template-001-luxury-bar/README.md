## 如何修改品牌名稱
搜尋所有 HTML 檔案中的 `LUXURY BAR` 和 `黑金餐酒館`，替換為您的品牌名稱。

## 如何替換圖片
1. 在 `assets/img/` 資料夾中放入您的圖片
2. 在 HTML 中找到對應的 `<img>` 標籤，修改 `src` 屬性
3. 或使用 Unsplash 圖片 URL 格式：`https://images.unsplash.com/photo-{ID}?w={width}&q=80`
