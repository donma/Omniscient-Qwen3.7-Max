# Fashion Boutique — 精品服飾網站模板 Prompt

## 1. 專案概述
建立一套黑白灰深色主題的精品服飾網站模板。

## 2. 設計風格
- 配色：黑白灰
- 字型：Bodoni+Moda + Noto Sans TC
- 版型：Masonry/Lookbook

## 3. 頁面結構
12 個頁面：首頁、品牌故事、服務項目、服務詳情、作品集、評價、FAQ、預約、流程、部落格、文章詳情、聯絡

## 4. 首頁 Hero
- 標題：定義風格的無限可能
- 副標：以極簡美學重新詮釋當代時尚，每一件單品都是對品質與品味的堅持。

## 5. 導航選單
首頁 / 品牌故事 / 商品分類 / Lookbook / 時尚專欄 / 聯絡我們

## 6. 服務項目
- 女裝系列：從日常到晚宴，精選來自歐洲與日本的設計師品牌，注重剪裁與面料的極致表現。
- 男裝系列：融合經典與當代設計語言，從商務正裝到街頭休閒，打造現代男性的多元面貌。
- 配件精選：手工皮件、絲巾、太陽眼鏡等精選配件，為您的造型畫龍點睛。
- 鞋履專區：從經典樂福鞋到前衛運動鞋，每一雙都經過嚴格品質篩選。
- 訂製服務：提供量身定制服務，由資深裁縫師一對一打造專屬於您的完美服飾。
- 造型顧問：預約專業造型顧問，根據您的身形、膚色與生活風格提供穿搭建議。

## 7. FAQ 內容
Q1：你們的品牌風格定位是什麼？
A：我們主打極簡主義與當代時尚的結合，精選來自全球頂尖設計師的作品。每件單品都經過嚴格篩選，確保品質與設計感兼具，適合追求獨特品味的都會人士。

Q2：有提供國際配送嗎？
A：是的，我們提供全球配送服務。台灣地區 1-2 天到貨，亞洲地區 3-5 天，歐美地區 7-10 天。訂單滿 NT$ 5,000 享免運優惠。

Q3：退換貨政策是什麼？
A：收到商品後 14 天內可辦理退換貨，商品需保持完整吊牌與包裝。訂製商品恕不退換，但可免費修改一次。

Q4：如何預約造型顧問？
A：您可以透過線上預約系統或致電門市預約。造型顧問服務完全免費，每次諮詢約 45-60 分鐘，建議提前一週預約。
