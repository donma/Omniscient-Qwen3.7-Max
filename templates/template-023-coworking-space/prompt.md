# Coworking Space — 共享空間網站模板 Prompt

## 1. 專案概述
建立一套白灰現代淺色主題的共享空間網站模板。

## 2. 設計風格
- 配色：白灰現代
- 字型：Space+Grotesk + Noto Sans TC
- 版型：Bento Grid

## 3. 頁面結構
12 個頁面：首頁、品牌故事、服務項目、服務詳情、作品集、評價、FAQ、預約、流程、部落格、文章詳情、聯絡

## 4. 首頁 Hero
- 標題：工作空間重新定義
- 副標：靈活、現代、充滿創意的共享工作空間，為自由職業者與新創團隊量身打造。

## 5. 導航選單
首頁 / 空間介紹 / 方案價格 / 設施服務 / 進駐企業 / 聯絡我們

## 6. 服務項目
- 熱桌方案：不固定座位，靈活使用開放工作區，適合自由職業者。
- 固定座位：專屬個人座位含置物櫃，享有穩定的工作環境。
- 獨立辦公室：2-10 人獨立辦公室，含門禁與訪客管理服務。
- 會議室租借：4-20 人會議室，含投影設備與白板，以小時計費。
- 活動場地：可容納 50-100 人的活動空間，適合講座與工作坊。
- 虛擬辦公室：商務地址、郵件收發與電話轉接服務。

## 7. FAQ 內容
Q1：可以免費試用嗎？
A：提供一日免費體驗（需預約），可體驗熱桌方案的所有設施。

Q2：有 24 小時進出嗎？
A：固定座位與獨立辦公室會員享有 24/7 進出權限。熱桌會員開放時間為 08:00-22:00。

Q3：包含哪些設施？
A：所有方案含高速 Wi-Fi、無限咖啡茶水、列印服務（月限 50 張）與公共區域使用。

Q4：可以帶寵物嗎？
A：小型寵物可進入指定區域，請保持安靜並確保不影響他人。
