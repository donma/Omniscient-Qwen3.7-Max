# Bookstore — 書店網站模板 Prompt

## 1. 專案概述
建立一套暖棕淺色主題的書店網站模板。

## 2. 設計風格
- 配色：暖棕
- 字型：Libre+Caslon+Text + Noto Sans TC
- 版型：Knowledge Hub

## 3. 頁面結構
12 個頁面：首頁、品牌故事、服務項目、服務詳情、作品集、評價、FAQ、預約、流程、部落格、文章詳情、聯絡

## 4. 首頁 Hero
- 標題：在書頁間發現世界
- 副標：不只是書店，更是知識與靈感的聚集地。嚴選好書、策展閱讀、舉辦講座。

## 5. 導航選單
首頁 / 本月選書 / 書籍分類 / 活動講座 / 閱讀空間 / 聯絡我們

## 6. 服務項目
- 文學小說：中外文學、類型小說與獨立出版品，每本經過店員用心選書。
- 人文社科：歷史、哲學、社會學等深度閱讀，拓展知識邊界。
- 藝術設計：建築、攝影、平面設計與美術類書籍，視覺與知識兼具。
- 童書繪本：精選國內外優質童書，培養孩子的閱讀興趣。
- 生活風格：料理、旅行、手作、園藝等生活類書籍。
- 雜誌期刊：國內外設計、文學、生活風格雜誌與季刊。

## 7. FAQ 內容
Q1：營業時間為何？
A：週一至週日 10:00-22:00，全年無休（除夕至初二除外）。

Q2：有會員制度嗎？
A：免費加入會員，消費集點可折抵書款。金卡會員享新書 85 折優惠。

Q3：可以線上購書嗎？
A：提供線上購書與宅配服務，雙北滿 NT$ 500 免運，其他地區滿 NT$ 800 免運。

Q4：有閱讀空間嗎？
A：店內設有閱讀區與咖啡區，消費飲品即可入座閱讀，不限時間。

## 9. 部落格文章
- 本月推薦：十本改變思維的書：編輯精選十本能顛覆你認知的好書，涵蓋科學、哲學與文學。
- 獨立書店的生存之道：在電商與連鎖的夾擊下，獨立書店如何找到自己的定位。
- 作家專訪：張曉風的書寫日常：與文學大師的午後對話，談創作、閱讀與生活。
- 如何建立閱讀習慣：每天 30 分鐘，養成終身受用的閱讀習慣的實用方法。
- 兒童閱讀指南：不同年齡段適合的書籍類型與親子共讀技巧。
- 書店空間改造紀實：我們如何將老倉庫改造成溫暖的閱讀空間。

## 10. 技術規格
- Bootstrap 5.3.3 CDN
- Bootstrap Icons 1.11.3 CDN
- Google Fonts CDN
- 純 HTML/CSS/JS
- file:// 環境可瀏覽

## 11. RWD 規範
三個斷點：992px / 768px / 576px

## 12. 圖片策略
使用 Unsplash 圖片來源，共 25 張

## 13. SEO 要求
每頁包含 meta tags、OG tags、Schema.org JSON-LD

## 14. 互動效果
漢堡選單、FAQ accordion、表單假送出、平滑捲動、回到頂部

## 15. 裝飾元素
page-curl 裝飾風格

## 16. Footer 設計
3 欄位 Footer，含品牌資訊、聯絡方式、快速連結

## 17. 表單設計
預約表單含姓名、電話、Email、日期、服務項目、備註欄位

## 18. 交付標準
每套模板 20 個檔案，所有文字繁體中文，可直接在瀏覽器開啟
