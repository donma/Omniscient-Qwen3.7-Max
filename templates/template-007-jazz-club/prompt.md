# Template-007 Jazz Club — 設計提示

## 概述
為爵士酒吧 / 現場音樂俱樂部設計的深色底復古網站。以黑金復古爵士為核心視覺語言，結合溫暖色調、懷舊排版與沉浸式影音氛圍。

## 色彩系統
- 主色：`#1a1208`（深棕黑）— 背景底色
- 強調色：`#c9a96e`（復古金）— 標題、按鈕、裝飾線
- 次要色：`#8b4513`（棕色）— 輔助區塊、卡片底
- 文字色：`#f5e6d3`（暖白）— 內文、說明文字

## 字型
- 標題：Noto Serif TC（700/900）
- 內文：Noto Sans TC（300/400/500）

## Hero 區域
- 全螢幕影音氛圍（CSS 深色漸層 + 動態粒子光點模擬）
- 居中 Logo 搭配左右選單（各 3 項）
- 底部 1px 金色細線

## 導航系統
- 桌機：居中 Logo，左右各 3 個選單項
- 手機：漢堡選單，右側滑入

## 卡片設計
- 復古邊框（雙層 border：外層金色 1px、內層棕色 1px）
- 暖色底 + 棕色文字

## 演出表
- 日期 + 演出者 + 曲風的列表式排版
- 金色日期標籤

## CTA
- 金色外框按鈕 + 文字連結
- Hover 時金色填充

## Footer
- 三欄：營業時間 / 社群連結 / 聯絡資訊
- 底部版權列

## 技術規格
- Bootstrap 5.3.3 CDN
- Bootstrap Icons 1.11.3 CDN
- Google Fonts: Noto Serif TC + Noto Sans TC
- 純前端靜態，file:// 可瀏覽
- RWD 三斷點（1200/992/768）
