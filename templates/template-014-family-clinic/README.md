# Template-014 Family Clinic — 家庭診所網站模板

## 概述
適合家庭診所、牙醫與醫療機構的白藍信任型專業網站模板。以白色為底搭配信任藍，營造乾淨、專業、值得信賴的醫療形象。

## 技術規格
| 項目 | 規格 |
|------|------|
| CSS 框架 | Bootstrap 5.3.3 CDN |
| 圖標庫 | Bootstrap Icons 1.11.3 CDN |
| 字型 | Source Sans 3 + Noto Sans TC |
| 主色 | #ffffff（白底） |
| 強調色 | #2563eb（信任藍） |
| 次要色 | #0ea5e9（淺藍） |
| 背景色 | #f0f7ff |
| 文字色 | #1e293b |

## 頁面列表（12 頁）
1. `index.html` — 首頁
2. `about.html` — 診所簡介
3. `services.html` — 科別介紹
4. `service-detail.html` — 科別詳情
5. `portfolio.html` — 設備環境
6. `reviews.html` — 病患評價
7. `faq.html` — 就醫 FAQ
8. `booking.html` — 線上預約
9. `process.html` — 就醫流程
10. `blog.html` — 健康知識
11. `blog-detail.html` — 文章詳情
12. `contact.html` — 聯絡我們

## 使用方式
直接用瀏覽器開啟 HTML 檔案即可瀏覽（支援 file:// 協定）。所有外部資源均使用 CDN，無需安裝任何套件。

所有圖片僅作為設計示意，正式交付前請替換為客戶自有或授權圖片。
