# Brunch Cafe — 早午餐網站模板 Prompt

## 1. 專案概述
建立一套奶油色淺色主題的早午餐網站模板。

## 2. 設計風格
- 配色：奶油色
- 字型：Playfair+Display + Noto Sans TC
- 版型：Magazine

## 3. 頁面結構
12 個頁面：首頁、品牌故事、服務項目、服務詳情、作品集、評價、FAQ、預約、流程、部落格、文章詳情、聯絡

## 4. 首頁 Hero
- 標題：美好的一天從早午餐開始
- 副標：用心食材、手作料理，在溫暖的空間裡享受最愜意的早午餐時光。

## 5. 導航選單
首頁 / 品牌故事 / 菜單一覽 / 空間環境 / 美食專欄 / 聯絡我們

## 6. 服務項目
- 經典早午餐：包含主餐、沙拉、湯品與飲品的豐盛套餐，開啟美好的一天。
- 手作甜點：每日限量手工甜點，從巴斯克蛋糕到季節水果塔。
- 精品咖啡：嚴選單品豆，手沖、虹吸等多種沖煮方式。
- 季節特餐：依當季食材推出限定菜單，呈現食材最鮮美的狀態。
- 下午茶套餐：三層甜點架搭配精選茶飲，最適合與好友共度午後時光。
- 客製派對：提供生日派對、寶寶派對等客製化佈置與菜單服務。

## 7. FAQ 內容
Q1：營業時間為何？
A：週二至週日 09:00-18:00，週一公休。最後點餐時間 17:00。

Q2：需要預約嗎？
A：建議週末預約，平日可現場排隊。4 人以上建議預約。可透過線上或電話預約。

Q3：有素食選項嗎？
A：有的，我們提供多道素食料理，包含全素與蛋奶素選項，菜單上有清楚標示。

Q4：寵物可以進入嗎？
A：戶外座位區歡迎寵物一同用餐，提供寵物飲水碗與小零食。
