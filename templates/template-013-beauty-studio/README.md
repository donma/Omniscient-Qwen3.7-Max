# T013 Beauty Studio — 奶茶色美容工作室模板

> 溫暖奶茶色調 × 左右分割 Split Layout × 大圓角卡片 — 專為美容工作室打造的專業網站模板。

## 模板特色

- **奶茶色溫暖色調**：#fdf6f0 淺底 + #c89b7b 奶茶棕 + #a67c52 深棕
- **左右分割 Hero**：左側文案 + 右側美容師工作照，視覺平衡
- **大圓角 20px**：所有卡片統一 20px 圓角，柔和親和
- **圓形圖標服務展示**：6 項服務搭配圓形 icon
- **完整 12 頁架構**：從首頁到聯絡，一應俱全
- **RWD 響應式**：支援桌面 / 平板 / 手機三斷點

## 頁面列表

| 11 | blog-detail.html | 文章詳情 |
| 12 | contact.html | 聯絡方式+地圖 |

## 技術規格

- **CDN**: Bootstrap 5.3.3 + Bootstrap Icons 1.11.3
- **字型**: Playfair Display (標題) + Noto Sans TC (內文)
- **結構化資料**: LocalBusiness Schema (BeautySalon)
- **互動**: 漢堡選單、FAQ Accordion、表單假送出、平滑滾動、Scroll Fade-up

## 使用方式

1. 直接以瀏覽器開啟 `index.html`
2. 圖片使用 Unsplash CDN，首次載入需網路連線
3. 可替換為自有圖片，更新 `assets/img/` 目錄
